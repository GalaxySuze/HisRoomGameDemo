# His Room
《TA的房间》是一款基于Godot 4.2引擎开发的文字推理小游戏演示，玩家通过点击场景内的道具触发对话，从中收集关键线索，并依据对话内容推断解答关于房间主人的相关谜题。**本版本为学习开发过程中的阶段性成果记录，仅呈现了游戏玩法的一个片段。**

# 关于美术
菜单与场景中的背景美术素材为个人手绘，请勿在任何商业环境中未经授权使用。
